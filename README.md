# Featured Events App 

## 📋 Project Overview
A modern, fully responsive event discovery platform built as part of the Dynamics 360 Web Developer case study. The application allows users to browse featured events, search for specific events, filter by category, and register for events.

## ✨ Features

### Core Requirements
- ✅ **Responsive Navigation Bar** - Logo and links to Home, Events, and Contact sections
- ✅ **Hero Section** - Eye-catching hero with "Discover Events Near You" heading
- ✅ **Featured Events** - 5 event cards displaying:
  - Event name
  - Date and time
  - Location
  - Short description
  - Register button with confirmation alert
- ✅ **Fully Responsive Design** - Optimized for mobile, tablet, and desktop devices
- ✅ **Clean UI** - Modern design using Bootstrap 5 with custom styling

### Bonus Features
- 🔍 **Real-time Search** - Filter events by name, description, or location
- 🏷️ **Category Filter** - Filter events by type (Art, Music, Food, Tech, Wellness)
- 📸 **High-quality Images** - Beautiful event photography from Unsplash
- ⚡ **Dynamic Data Loading** - Simulated API fetch with loading spinner
- 🎨 **Modern Navbar** - Scroll effects, glass morphism, active section highlighting
- 📊 **Event Statistics** - Shows event counts and attendee numbers
- 🎯 **Smooth Scrolling** - Navigation links smoothly scroll to sections
- 📱 **Mobile Optimized** - Hamburger menu for mobile devices

## 🛠️ Technologies Used

- **React 18** - Frontend framework with functional components and hooks
- **Bootstrap 5** - Responsive grid system and UI components
- **CSS3** - Custom animations, gradients, and modern styling
- **React Hooks** - useState and useEffect for state management
- **Vite** - Fast build tool and development server
- **Unsplash API** - High-quality event photography
